# OWASP-Juice-Shop-VAPT-Assessment-
Performed a web application security assessment on OWASP Juice Shop using Burp Suite to analyze authentication flows, JWT handling, API security, access control weaknesses, IDOR vulnerabilities, and HTTP security configurations within a controlled lab environment.
Overview

This project demonstrates a structured web application security assessment performed on OWASP Juice Shop using Burp Suite within a controlled local lab environment.

The assessment focused on:

Authentication analysis

JWT token analysis

API security testing

Access control validation

IDOR testing

HTTP security header assessment

OWASP Top 10 mapping



Project Objectives

Capture and inspect HTTP traffic using Burp Suite

Analyze authentication requests and JWT handling

Identify sensitive information disclosure

Test API authorization mechanisms

Perform IDOR (Insecure Direct Object Reference) testing

Evaluate HTTP security headers

Validate JWT signature verification behavior

Document findings aligned with OWASP Top 10



Tools & Technologies Used

OWASP Juice Shop

Burp Suite Community Edition

JWT.io

Kali Linux

Chromium Browser

HTTP/JSON APIs
